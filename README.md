# 🚀 AI Caption Generator

A sleek AI-powered web app that generates Instagram-style captions from uploaded images.

---

## 🌐 Live Demo

👉 https://caption-generator-w4zr.onrender.com

---

## ✨ Features

* 📸 Upload any image
* 🤖 AI generates creative captions
* 🎨 Modern dark UI with neon glow
* 📋 One-click copy caption

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* OpenRouter API
* ImgBB API

---

## ⚙️ Setup

```bash
git clone https://github.com/your-username/caption-generator.git
cd caption-generator
pip install -r requirements.txt
python app.py
```

---

## 🔑 Environment Variables

Create a `.env` file:

```
OPENROUTER_API_KEY=your_key
IMGBB_API_KEY=your_key
```
